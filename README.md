# Integrating Graphs, Large Language Models, and Agents: Reasoning and Retrieval

Official repository accompanying the survey:

**Integrating Graphs, Large Language Models, and Agents: Reasoning and Retrieval**

## Overview

This repository provides a continuously updated collection of resources related to Graph–LLM integration, GraphRAG systems, knowledge graph reasoning, graph-enhanced retrieval, Graph-Agent-LLM architectures, and multimodal graph reasoning.

The goal of this repository is to support researchers and practitioners by providing:

* A curated taxonomy of Graph–LLM methods
* Links to representative papers
* Open-source implementations
* Datasets and benchmarks
* Emerging Graph-Agent-LLM frameworks

## Taxonomy

The survey organizes the field into the following major categories:

1. LLM-Assisted Graph Construction
2. Graph-Enhanced LLM Reasoning
3. Hybrid GNN–LLM Architectures
4. Knowledge Graph Question Answering (KGQA)
5. Scene Graph Integration
6. Graph-Agent-LLM Systems
