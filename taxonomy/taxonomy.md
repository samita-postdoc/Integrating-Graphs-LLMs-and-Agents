# Graph–LLM Taxonomy

This directory contains the taxonomy and conceptual framework introduced in the survey:

**Integrating Graphs, Large Language Models, and Agents: Reasoning and Retrieval**

## Overview

The taxonomy organizes the rapidly growing Graph–LLM ecosystem into several major research directions:

1. LLM-Assisted Graph Construction
2. Graph-Enhanced Reasoning
3. Hybrid GNN–LLM Architectures
4. Knowledge Graph Question Answering (KGQA)
5. Scene Graph Integration
6. Graph-Agent-LLM Systems

The goal of this taxonomy is to provide a unified perspective on how graph representations, large language models, and agentic systems interact to support retrieval, reasoning, planning, and decision-making.
