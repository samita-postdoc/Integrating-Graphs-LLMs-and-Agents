# Knowledge Graph Question Answering (KGQA)

## Overview

Knowledge Graph Question Answering (KGQA) focuses on answering natural language questions using structured knowledge graphs. Recent Graph-LLM approaches integrate large language models with knowledge graphs to improve entity linking, reasoning, retrieval, and answer generation.

## Research Directions

* Traditional KGQA
* Neural KGQA
* LLM-enhanced KGQA
* Retrieval-augmented KGQA
* Multi-hop reasoning
* Explainable KGQA

## Common Benchmarks

* WebQSP
* ComplexWebQuestions (CWQ)
* MetaQA
* LC-QuAD
* QALD
* KQA Pro

## Representative Papers

* StructGPT
* KG-Agent
* Knowledge Graph-Augmented Language Models
* Graph Retrieval for Question Answering
* LLM-based KGQA Frameworks
